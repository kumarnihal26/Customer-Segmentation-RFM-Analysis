## Dataset Information

The dataset used in this project is the Online Retail II dataset.

Due to file size limitations, the full CSV file is not uploaded to GitHub.

Dataset Source: https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci?resource=download
UCI Machine Learning Repository / Kaggle (Online Retail II)

Users can download the dataset directly from the source and place it in this folder
to reproduce the analysis.
