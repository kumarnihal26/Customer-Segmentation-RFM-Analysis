# Customer-Segmentation-RFM-Analysis
Customer segmentation using RFM analysis on retail transaction data

## Project Overview
This project applies RFM (Recency, Frequency, Monetary) analysis to segment customers
based on their purchasing behavior using real-world retail transaction data.

## Business Objective
To help marketing teams identify:
- Best customers
- Loyal customers
- At-risk customers
- Customers needing attention

## Dataset
Online Retail II Dataset (UCI Machine Learning Repository)

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Key Insights
- Majority of customers fall under the "Need Attention" segment
- Loyal and Best Customers contribute significantly to revenue
- At-risk customers can be targeted with re-engagement campaigns

## Conclusion
RFM analysis provides actionable insights for data-driven marketing strategy
and customer relationship management.
